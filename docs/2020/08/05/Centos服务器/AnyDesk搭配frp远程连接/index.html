AnyDesk搭配frp远程连接

发表于 2020-08-05 | 分类于 centos

服务器frps

  • 安装frp服务端

  1. cd /usr/local进入/usr/local目录
  2. wget https://github.com/fatedier/frp/releases/download/v0.29.0/frp_0.29.0_linux_amd64.tar.gz下载frp服务器压缩包
  3. tar xzvf frp_0.29.0_linux_amd64.tar.gz解压压缩包
  4. cd frp_0.29.0_linux_amd64进入frp目录，当前目录为/usr/local/frp_0.29.0_linux_amd64

网络拓扑

网络拓扑理解如下，受控客户端的AnyDesk服务在本地监听7070端口，frp服务将其映射到云服务器的7080端口。控制客户端连接云服务器的7080端口，就会被frp转发到受控客户端的7070端口，此时受控客户端的AnyDesk收到连接请求，就这样实现远程桌面内网的穿透。
