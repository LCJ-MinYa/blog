```js
let n = 3 // digits u want
let iterator = {
  [Symbol.iterator]: function*() {
    let num = 0
    let result = ''
    for(;;){
      result = num.toString('36')
      if(result.length > n) break
      yield result
      num += 1
    }
  }
}

let arr = [...iterator]
console.log(arr)
```

## 知识点
* <code>for(;;){}</code>表示无限循环,和<code>while(true){}</code>功能一样.
* <code>num.toString('36')</code>,36进制，可选参数.
```js
1.toString('36') // 'a'
2.toString('36') // 'b'
....
```
