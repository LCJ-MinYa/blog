## 提交到本地
* <code>git add .</code>

## 提交到仓库
* <code>git commit -m 'xxx'</code>

## 查看当前git状态
* <code>git status</code>

## 放弃本地文件
* <code>git checkout fileName</code>放弃fileName文件修改
* <code>git checkout .</code>放弃所有本地文件修改

## 拉取更新
* <code>git pull</code>
